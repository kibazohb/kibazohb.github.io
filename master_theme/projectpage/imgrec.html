IMAGE RECREATION WITH GENETIC ALGORITHM.

The Project

This is the culmination project for my course 2S03 - Programming in C, at McMaster University. This project put my programming skills and memory allocation knowledge of C to the test.

The instructor introduced the class to genetic algorithms, and it was applied on the project. The algorithm begins by creating a random initial population. The algorithm then creates a sequence of new populations. At each step, the algorithm uses the individuals in the current generation to create the next population. In this case, the individuals were a group of pixels. The result is as below. (hover over the card to see)
